# Next.js + Tailwind CSS Google Frontend clone

This project is a frontend clone of google built with Next JS and Tailwind CSS.
It uses Google Programmable Search API and the new [`Just-in-Time Mode`](https://tailwindcss.com/docs/just-in-time-mode) for Tailwind CSS.

# Preview
  https://next-js-google-clone-one.vercel.app/

# Prerequisites

Google search engine api key: https://developers.google.com/custom-search/v1/using_rest

Google programmable search Context key: https://cse.google.com/cse/create/new


## How to use

Clone the repo:
    git clone https://github.com/Slick-RaphaEl/chop-beta
    
Install NPM packages:
    npm install
